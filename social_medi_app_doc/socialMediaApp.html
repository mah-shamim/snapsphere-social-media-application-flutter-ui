1) Project Version - top

Please use the following flutter and dart version for this project.

  • Flutter 3.3.0 - Channel stable
  • Dart 2.18.0

2) Open  Project - top

Open android studio and click on Open button in top-right corner of the screen.

Browse to the Main Directory of the project and select open option.

3) Project Structure - top

From the left side of screen, open "lib" folder once the Android Studio completes updating necessary files.

 

4) Change Application Name - top

For Android:

To change the app name in the manifest file of an Android Studio project, you need to modify the android:label attribute of the or tag in the manifest file.

For iOS:

To change application name in iOS, right click on the iOS directory and then simply go to plist and simply replace the value of CFBundleDisplayName

       

5) Reskin Project Images - top

For Android:

To change and update the icons, select the "android" folder and then go to "app" directory. From there navigate to "src" folder and then move to "main" folder. Now, open the "res" folder.

There are two types of folders here.

1. Mipmap (hdpi, mdpi,......, xxxhdpi)

Updates the icon inside the mipmap to change main application launcher icon and splash screen.

2. Drawable (hdpi, mdpi,......, xxxhdpi)

Updates the drawable icon which will affect the notification icons.

For IOS:

To change and update the icons, select the iOS folder and then go to Runner directory. From there navigate to Assets.xcassets folder and open it with Finder.

6) Change colour - top

To change the colour anywhere in the application, you can change the colour code in the "colors_utils.dart" file.

 

7) Update Directory or File Name - top

To change the name of package or a file, right click on the package or file you want to update and select "Refactor" and then click on "Rename" option.

 

8) Change Application Id - top

For Android:

To change Application Id for android, Navigate to app folder which is inside the android directory and then open the src folder. Now, open "build.gradle" file.

Now, move down to the "defaultConfig" block in the file and over there you can change the highlighted text with your application id.

       

For iOS:

To change bundle identifier in iOS, Open Info.plist file and change the BundleIdentifier with your required Identifier for IOS. Here below is the image."
